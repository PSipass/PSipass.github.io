Roomify

January 2026

Project Overview

Roomify is an application made for the class Computational Creativity. Being tasked with developing a tool of sorts which in some capacity allows the user to create something with the help of a machine (be it a simple algorithm or a sophisticated LLM), we developed a room-planning application. Roomify allows the user to place a wide array of furniture on a 2D representation of room and work together with an AI model to not only optimize furniture placement, but also discover new, alternative methods of arranging furniture. To get a clearer picture of the feel of the room with the furniture, the AI model can also be tasked with making a 3D representation of the room from any angle the user desires. Furthermore, using some sliders the user can instruct the model to change the look of the furniture to fit different interior design styles from around the world such as Scandinavian, Middle Eastern, and Japanese.

Made with: Mats Doornbos, Timothy Schuur
