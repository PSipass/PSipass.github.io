Custom DAW/Sampler

January 2025

Project Overview

Flexing each and every coding muscle I built up from scratch in the course Creative Programming, I developed an interactive digital chiptune music tool. Equipped with four different waveforms at a four-octave range and an 8-bit videogame inspired drumpad, users can play either iconic melodies or create completely original music. Its standout feature is a sampling function which allows users to upload any mp4 file to include in their performance. The sample can be trimmed, of course, but with the inclusion of a pitch slider and a low- and high-pass filter even more possibilities are provided.
